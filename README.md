# transrepjam2026

Store Simulator
video games / home videos / card games / internet cafe / cafe / ttrpgs

Build different parts of the store out
Most customers just walk up with "I want this" and you serve them
Some customers appear in the store with little storylines
Talk to them with morrowind-like dialogue - click various words in their text to unlock more options
Some of them are baby trans and it's very cute
If you help them they start coming in with some self care - happier selves.
"Speedrun to crack X egg in 30 days" joke here

The rest of the game is a standard shop owner simulator - place shelves, stock them, place things the customers can pay to use etc etc.
Having certain things in your store will open certain storylines

Primary objective of the game is to have nice store.
Primary objective of the game is also to explore the storylines and help people.

Some "vignettes" of the player character showing their journey(s) can be unlocked
